# Cloud Computing Concepts, Technology, Security, and Architecture (2nd Edition)

**Type**: RAG stub — full content in vector index, not in wiki

## Abstract

This is the second edition of Thomas Erl and Eric Barceló's comprehensive, vendor-neutral textbook on cloud computing, published by Pearson in 2023 as part of the Pearson Digital Enterprise Series. The book systematically covers fundamental cloud computing concepts, cloud infrastructure and specialized mechanisms, cybersecurity mechanisms, cloud architecture patterns, delivery and deployment models, cost metrics, pricing models, SLAs, and emerging topics such as edge and fog computing and containerization (Docker, Kubernetes). It aims to provide industry-grounded clarity that empowers readers to distinguish genuine cloud-related offerings from marketing and to make informed business and technology decisions.

## Key Claims

- A vendor-neutral understanding of cloud computing is essential for filtering the cloud product and service marketplace to focus on what genuinely serves business requirements
- Cloud computing can be decomposed into fundamental concepts, delivery models, deployment models, defined roles, and clear boundaries
- Cloud infrastructure relies on specific mechanisms including logical network perimeters, virtual servers, hypervisors, cloud storage devices, cloud usage monitors, resource replication, ready-made environments, and containers
- Cybersecurity in cloud environments is addressed through discrete access-oriented mechanisms such as encryption, hashing, digital signatures, cloud-based security groups, PKI, SSO, and hardened configurations
- Cloud architectures encompass specialized patterns including storage maintenance window architecture, edge computing architecture, and fog computing architecture

## Topics Covered

Fundamental cloud computing concepts, cloud delivery models (IaaS/PaaS/SaaS), cloud deployment models, cloud infrastructure mechanisms (virtual servers, hypervisors, containers), specialized cloud mechanisms (auto-scaling, load balancing, failover, resource clustering), cloud cybersecurity mechanisms (encryption, PKI, SSO, digital signatures), cloud architecture patterns, edge and fog computing, cost metrics and pricing models, service quality metrics and SLAs, containerization technologies (Docker, Kubernetes), case studies (ATN, DTGOV, Innovartus)
