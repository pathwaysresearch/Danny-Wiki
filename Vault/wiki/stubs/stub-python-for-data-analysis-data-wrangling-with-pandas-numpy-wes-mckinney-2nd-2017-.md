## Abstract

This is the second edition (October 2017) of Wes McKinney's comprehensive guide to data analysis in Python, updated for Python 3.6. Written by the creator of the pandas library, it provides complete instructions for manipulating, processing, cleaning, and crunching datasets using pandas, NumPy, IPython, and Jupyter. The book serves as a practical, modern introduction to data science tools in Python, packed with hands-on case studies targeting both analysts new to Python and Python programmers new to data science and scientific computing.

## Key Claims

- Python's pandas, NumPy, IPython, and Jupyter ecosystem forms a powerful toolkit for solving a broad set of data analysis problems effectively
- The book provides complete instructions for manipulating, processing, cleaning, and crunching datasets in Python
- The second edition has been updated for Python 3.6 and covers the latest versions of pandas, NumPy, IPython, and Jupyter
- Practical case studies are used throughout to demonstrate real-world data analysis workflows
- The book is designed to serve both analysts new to Python and experienced Python programmers new to data science
