# Practical Statistics for Data Scientists: 50+ Essential Concepts Using R and Python

**Type**: RAG stub — full content in vector index, not in wiki

## Abstract

This is the second edition of O'Reilly's *Practical Statistics for Data Scientists* by Peter Bruce, Andrew Bruce, and Peter Gedeck (2020). It covers 50+ essential statistical concepts specifically oriented toward data science practitioners, with implementations in both R and Python. The book spans exploratory data analysis, statistical experiments, regression, classification, unsupervised learning, and related topics, bridging classical statistics and modern machine learning methodology. It is a widely referenced practitioner's guide that emphasizes which statistical methods are most relevant to data science workflows.

## Key Claims

- Statistical concepts should be taught with direct relevance to data science practice, distinguishing what matters from what is merely traditional
- Resampling methods (e.g., permutation tests, bootstrap) provide practical alternatives to classical chi-square and parametric tests for hypothesis testing
- Regression and prediction require careful attention to model selection, cross-validation, weighted regression, and the dangers of extrapolation
- Multi-arm bandit algorithms offer a data-science-relevant alternative to traditional A/B testing frameworks for experimental design
- Unsupervised learning techniques such as PCA, correspondence analysis, and K-means clustering require principled approaches to interpreting components and selecting the number of clusters

## Topics Covered

Exploratory data analysis and structured data, statistical experiments and significance testing, resampling and bootstrap methods, chi-square and Fisher's exact tests, multi-arm bandit algorithms, power and sample size, simple and multiple linear regression, model selection and stepwise regression, cross-validation, prediction and confidence intervals, principal component analysis, K-means clustering and cluster selection
