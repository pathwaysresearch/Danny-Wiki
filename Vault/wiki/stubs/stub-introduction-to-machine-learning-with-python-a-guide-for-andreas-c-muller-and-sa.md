## Abstract

This is the first edition (October 2016) of O'Reilly's practical guide to machine learning using Python and the scikit-learn library, authored by Andreas C. Müller and Sarah Guido. The book teaches readers — including Python beginners — how to build their own machine learning solutions for commercial applications and research projects. It covers the full workflow of creating successful machine learning applications, emphasizing that the field is accessible beyond large companies with extensive research teams. The book is structured as a comprehensive, hands-on guide leveraging the abundance of available data and the scikit-learn ecosystem.

## Key Claims

- Machine learning has become integral to many commercial applications and research projects, but is not exclusive to large companies with extensive research teams
- Python users, even beginners, can learn practical ways to build their own machine learning solutions
- The scikit-learn library is the primary tool used throughout the book for implementing machine learning
- Machine learning applications are limited only by imagination given the volume of data available today
- The book covers the necessary steps to create a successful end-to-end machine learning application

## Topics Covered

Supervised learning, unsupervised learning, scikit-learn API, data representation for machine learning, model evaluation, feature engineering, Python for data science, practical machine learning workflows, classification, regression, clustering, model selection
