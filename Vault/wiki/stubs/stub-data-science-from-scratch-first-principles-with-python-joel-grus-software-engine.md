# Data Science From Scratch: First Principles With Python — Joel Grus

**Type**: RAG stub — full content in vector index, not in wiki

## Abstract

This book by Joel Grus (published by O'Reilly Media, 2015) teaches data science fundamentals by building tools and algorithms from scratch in Python rather than relying on pre-built libraries. The author focuses on developing both the hacking skills and the mathematical/statistical foundations at the core of data science. Implementations are deliberately built for clarity and learning on small toy data sets, with pointers to production-ready libraries for real-world use. The book advocates Python as the best language for learning data science due to its simplicity, free availability, and rich ecosystem of data science libraries.

## Key Claims

- The best way to learn data science hacking skills is by hacking on things — building implementations yourself rather than only using existing tools
- Python is the obvious and best choice for learning data science over R, Java, or Scala
- Python is well-suited for data science because it is free, relatively simple to code in, and has many useful data science-related libraries
- The tools built from scratch in the book are illuminating but deliberately impractical — they work on small data sets but not at "web scale"
- Mathematics and statistics are at the core of data science and require deep comfort, not just surface familiarity

## Topics Covered

Linear algebra, statistics, probability, machine learning algorithms from scratch, Python programming fundamentals, data visualization, working with data, gradient descent, hypothesis and inference, k-nearest neighbors, decision trees, neural networks, natural language processing, clustering, recommender systems, databases and SQL, MapReduce, network analysis, naive Bayes, regression
