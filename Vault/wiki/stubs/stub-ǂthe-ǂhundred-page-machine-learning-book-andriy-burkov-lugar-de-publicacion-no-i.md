# The Hundred-Page Machine Learning Book — Andriy Burkov

**Type**: RAG stub — full content in vector index, not in wiki

## Abstract

This is Andriy Burkov's concise yet comprehensive introduction to machine learning, covering the field from foundational concepts through fundamental algorithms, practical engineering considerations, and advanced topics in roughly one hundred pages. The book frames machine learning as the process of finding mathematical formulas that generalize from training data to unseen inputs drawn from similar statistical distributions, explicitly noting that "machines don't learn" in the human sense. It spans supervised, unsupervised, and reinforcement learning paradigms, and includes coverage of neural networks, deep learning, support vector machines, ensemble methods, and practical advice on feature engineering and model evaluation. Distributed on a "read first, buy later" principle, it is supplemented by a companion wiki (theMLbook.com) with additional resources.

## Key Claims

- Machine learning is fundamentally about finding mathematical formulas that produce desired outputs from training data and generalize to new inputs from similar statistical distributions
- "All models are wrong, but some are useful" (George Box) — the book adopts a pragmatic, applied perspective on modeling
- Learning types include supervised, semi-supervised, unsupervised (clustering, dimensionality reduction, outlier detection), and reinforcement learning
- Slight distortions to inputs can break a trained model's outputs, distinguishing ML from true human-like learning
- The book is structured to help practitioners determine whether ML can solve their problem and which techniques to try

## Topics Covered

Machine learning fundamentals, supervised learning, unsupervised learning (clustering and dimensionality reduction and outlier detection), linear regression, logistic regression, decision tree learning, support vector machines, k-nearest neighbors, gradient descent and learning algorithm anatomy, feature engineering, neural networks and deep learning, shallow vs. deep learning, model-based vs. instance-based learning, parameter estimation, genetic algorithms, reinforcement learning, ensemble methods, basic ML practice
